Standardized templates for reporting statistical outcomes (APA 7th Edition)

Categories

APA 7th Rules

  • Italicize statistical letters (M, SD, t, p, F, r).
  • Do not use a leading zero for numbers that cannot exceed 1 (e.g., p = .04, not 0.04).
  • Report exact p values to two or three decimal places.
  • If p is less than .001, write p < .001.

Terminology Guide

Parametric Tests: Assume underlying statistical distributions (e.g., normal distribution) and usually analyze means (e.g., t-test, ANOVA). Best for continuous, normally distributed data.

Non-parametric Tests: "Distribution-free" tests that don't assume a normal distribution. Often analyze medians or ranks (e.g., Mann-Whitney U, Kruskal-Wallis). Best for skewed data, small sample sizes, or ordinal data.

Chi-Square Test of Independence

When to use

Determining whether there is an association between two categorical variables.

APA Format Template
A chi-square test of independence was performed to examine the relation between [Categorical Variable 1] and [Categorical Variable 2]. The relation between these variables was significant, X²([df], N = [sample size]) = [chi-square value], p = [p-value].
Written Example
A chi-square test of independence was performed to examine the relation between voting preference and gender. The relation between these variables was significant, X²(2, N = 300) = 14.52, p = .001.

Rule: Use the Greek letter Chi squared (χ² or X²). Include N inside the parentheses.

Fisher's Exact Test

When to use

Alternative to Chi-Square for categorical data when sample sizes are small (expected cell counts < 5).

APA Format Template
A Fisher's exact test was conducted to determine if there was a significant association between [Variable 1] and [Variable 2]. There was a statistically significant association between the variables, p = [p-value].
Written Example
A Fisher's exact test was conducted to determine if there was a significant association between treatment group and adverse side effects. There was a statistically significant association between the variables, p = .021.

Rule: Fisher's exact test does not have a formal test statistic like chi-square, so simply report the exact p-value.

Mann-Whitney U Test

When to use

Non-parametric alternative to the independent t-test (comparing medians of two independent groups).

APA Format Template
A Mann-Whitney U test was run to determine if there were differences in [dependent variable] scores between [Group 1] and [Group 2]. Median scores were statistically significantly higher in [Group 1] (Mdn = [Median 1]) than in [Group 2] (Mdn = [Median 2]), U = [U-value], p = [p-value].
Written Example
A Mann-Whitney U test was run to determine if there were differences in pain tolerance scores between Group A and Group B. Median scores were statistically significantly higher in Group A (Mdn = 8.5) than in Group B (Mdn = 5.0), U = 120.5, p = .023.

Rule: Report Medians (Mdn) instead of Means (M). Report the U statistic.

Wilcoxon Signed-Rank Test

When to use

Non-parametric alternative to the paired t-test (comparing medians of two related/paired samples).

APA Format Template
A Wilcoxon signed-rank test was conducted to evaluate whether [dependent variable] differed significantly between [Time 1] and [Time 2]. The results indicated a significant [increase/decrease] in scores from [Time 1] (Mdn = [Median 1]) to [Time 2] (Mdn = [Median 2]), T = [T-value], z = [z-value], p = [p-value].
Written Example
A Wilcoxon signed-rank test was conducted to evaluate whether anxiety scores differed significantly between pre- and post-intervention. The results indicated a significant decrease in anxiety scores from pre-intervention (Mdn = 45.0) to post-intervention (Mdn = 32.0), T = 15.5, z = -2.85, p = .004.

Rule: Report Medians (Mdn) and the test statistic (T or z depending on sample size).

🏔️

Kruskal-Wallis H Test

When to use

Non-parametric alternative to One-Way ANOVA (comparing medians of 3+ independent groups).

APA Format Template
A Kruskal-Wallis H test was conducted to determine if there were differences in [dependent variable] between [Group 1], [Group 2], and [Group 3]. Median [dependent variable] scores were statistically significantly different between the groups, H([df]) = [H-value], p = [p-value].
Written Example
A Kruskal-Wallis H test was conducted to determine if there were differences in pain levels between the medication, physical therapy, and control groups. Median pain scores were statistically significantly different between the groups, H(2) = 11.84, p = .003.

Rule: The test statistic is typically reported as H or χ².
